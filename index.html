How It Works

Three phases. One goal.

TraderAI is organized into three clean, separate stages. Each one has a single job — so the system stays easy to understand and easy to trust.

PHASE 01

Configuration

Runs once, at startup. You set three parameters — investment amount, order mode, and signal confirmation — and then the bot takes it from there for the whole session.

PHASE 02

Trading Loop

The bot fetches a fresh AI-generated watchlist every hour and evaluates every stock on it every 15 seconds, producing a Buy, Sell, or Hold signal for each one.

PHASE 03

Execution

Any signal that passes the bot's checks is sent to Alpaca via their trading API. The order is placed automatically — no human action needed.

The Operational Loop

What happens every 15 seconds

Once running, the bot follows a strict schedule automatically. Here's the exact sequence it repeats, over and over, until you stop it.

01

Watchlist Generation (hourly)

Every hour, Gemini 2.5 Flash returns 5 stock picks each tagged with a risk level — high, medium, or low. That classification shapes how aggressively the bot acts on each signal.

02

Signal Evaluation (every 15s)

The algorithm scans every stock on the current watchlist and issues a Buy, Sell, or Hold. It tracks open positions too, so it never doubles up on the same stock.

03

Optional Trend Confirmation

If enabled, any Buy or Sell signal is cross-checked against the price trend from the past 30 seconds. Both must agree — or the bot holds and waits for the next tick.

04

Order Submission

Signals that clear all checks are sent to Alpaca. In algorithmic mode, the bot calculates and attaches a stop-loss and take-profit price alongside every order it submits.

Configuration

You set the rules. Once.

Before the bot starts trading, it asks you three questions. Your answers define how it behaves for the entire session.

investment_amount
Investment Amount Per Trade

The fixed amount of capital the bot commits to each individual position it opens. This is your primary lever over exposure — every single trade, for every stock, uses this exact figure.

order_mode
Order Pricing Mode

Choose between market orders (execute at current price instantly) or algorithmic mode, where the bot calculates an entry price, stop-loss, and take-profit target for every trade automatically.

confirm_signals
Signal Confirmation Mode

When enabled, the bot adds a second check: it looks at the price trend over the last 30 seconds before acting. Both the primary algorithm and the trend must agree, or the bot holds. This trades frequency for confidence.

Get Started

Up and running in minutes.

TraderAI runs on Python and connects to Alpaca's free paper trading account. Here's everything you need.

01

Install Python & VS Code

Download Python from python.org and install the Python extension in Visual Studio Code for the best experience.

02

Install the required packages

Open your terminal and run this command:

pip install alpaca-py pandas numpy google-genai
03

Download & unpack TraderAI

Download the ZIP file below, extract it, and open the folder in VS Code.

04

Add your Gemini API Key

Open the file Stock_Watchlist_Gemini in VS Code. Press CTRL + F and search for ADD_KEY_HERE — replace it with your own Gemini API Key, which you can get for free at aistudio.google.com.

05

Create a free Alpaca account

Sign up at alpaca.markets, navigate to your dashboard, and copy your API Key and Secret Key from the paper trading section.

06

Run TraderAI & enter your keys

Launch the bot — it will prompt you for your API Key and Secret Key on startup. Paste them in, set your three configuration options, and TraderAI will do the rest.
